QUICK START GUIDE
================================================================================

1. EXTRACT THE APPLICATION
   - Extract FundiMzii-App to C:\xampp\htdocs (Windows)
   - Or /opt/lampp/htdocs (Linux/Mac)

2. CREATE DATABASE
   - Open http://localhost/phpmyadmin
   - Create database: fundimzii

3. OPTIONAL: INSTALL COMPOSER
   - Run: composer install
   - Run: php artisan migrate
   - Run: php artisan db:seed (for sample data)

4. ACCESS APPLICATION
   - Open: http://localhost/FundiMzii-App/public
   - Or: http://localhost:8000 (if using php artisan serve)

5. START USING
   - Click "New Client" to register your first tailor profile
   - Add measurements for clients
   - Create orders and track status
   - View dashboard for business insights
   - Export measurements to PDF
